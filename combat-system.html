Боевая система

Здесь представлены скрипты для боевой системы. Указано, куда их вставлять и какой это тип скрипта.

1. Скрипт для атаки

Тип скрипта: ServerScript

Куда вставлять: Внутрь модели игрока (в папку StarterCharacterScripts)

            
// ServerScript для атаки
local function Attack(player)
    print(player.Name .. " атакует!")
end

2. Локальный скрипт для управления игроком

Тип скрипта: LocalScript

Куда вставлять: Внутрь папки StarterPlayerScripts

            
// LocalScript для управления
local player = game.Players.LocalPlayer
local function OnKeyPress(input)
    if input.KeyCode == Enum.KeyCode.E then
        print("Игрок атакует!")
    end
end
game:GetService("UserInputService").InputBegan:Connect(OnKeyPress)
